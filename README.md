# 📘 คู่มือการใช้งาน Driver สำหรับ Mini Keyboard 6 ปุ่ม + 1 Knob (เวอร์ชันอัปเกรด)

## 🎯 คำอธิบายโครงการ
โปรเจกต์นี้เป็น **driver setup สำหรับ Mini Keyboard** ที่มี **6 ปุ่มและ 1 knob** รองรับการตั้งค่าแมโครและการควบคุมผ่านเว็บบราวเซอร์โดยใช้ **WebHID** พร้อม **Export** เป็นสคริปต์ **AutoHotkey v2** เพื่อใช้งานบน Windows. UI ได้รับการออกแบบใหม่ด้วยธีม Light/Dark, ฟอนต์ Noto Sans Thai, Tooltip ภาษาไทย, Drag & Drop, และการควบคุม LED Backlight.

โดยได้รับการออกแบบใหม่ทั้งหมดเพื่อให้ UX สวยงามและใช้งานง่าย:
- ธีม **Light / Dark** สไตล์โมเดิร์น (glassmorphism) 
- ฟอนต์ **Noto Sans Thai** พร้อม **การแสดงผลภาษาไทยเต็มรูปแบบ** 
- **Tooltip** แสดงคำอธิบายเป็นไทยเมื่อชี้เมาส์หรือคลิก 
- **Drag & Drop** สำหรับจัดเรียงแมโครและตั้งค่าแอคชัน 
- ตัวควบคุม **LED Backlight** พร้อมการแสดงผลแบบ realtime 
- รองรับการบันทึกและเล่นเสียงสวิตช์เครื่องจักรจำลองด้วย **Web Audio API** 

## 📥 การติดตั้งและเริ่มต้นใช้งาน
1. **ดาวน์โหลดโค้ด**  
   - Clone repository หรือดาวน์โหลดไฟล์ ZIP แล้วแตกไว้ในโฟลเดอร์ที่ต้องการ.
2. **เปิดโปรเจกต์**  
   - เปิดไฟล์ `index.html` ด้วยเว็บบราวเซอร์ที่สนับสนุน WebHID (Chrome, Edge).  
3. **เชื่อมต่อคีย์แพด**  
   - กดปุ่ม `Connect` แล้วเลือกอุปกรณ์จากรายการที่แสดง.  
4. **กำหนดแมโคร**  
   - ลากไอคอนแอคชันจาก **Palette** ไปที่ปุ่มที่ต้องการบนคีย์แพด.  
   - คลิกที่ปุ่มเพื่อเปิด **Editor** ปรับค่าต่าง ๆ (delay, repeat, ฯลฯ).  
5. **บันทึกและส่งออก**  
   - กด `Save Config` เพื่อบันทึกเป็นไฟล์ JSON ในเครื่องของคุณ.  
   - กด `Export to AHK` เพื่อสร้างสคริปต์ `macro.ahk` (AutoHotkey v2) ที่สามารถรันบน Windows ได้.

## ✨ ฟีเจอร์หลัก
| ฟีเจอร์ | รายละเอียด |
|---|---|
| **ธีม Light/Dark** | ปรับได้จากเมนู Settings → Theme |
| **ภาษาไทย** | ทุกส่วนของ UI แสดงผลเป็นภาษาไทย พร้อมฟอนต์ Noto Sans Thai |
| **Tooltip** | แสดงคำอธิบายสั้น ๆ เมื่อวางเมาส์บนปุ่มหรือไอคอน |
| **Drag & Drop** | จัดเรียงและเพิ่มแอคชันโดยการลากจาก Palette ไปยังปุ่ม |
| **LED Visualizer** | แสดงสถานะและความสว่างของ LED บน UI แบบ realtime |
| **เสียงสวิตช์** | จำลองเสียงกดสวิตช์ (Blue, Brown, Red) ด้วย Web Audio API |
| **WebHID Integration** | เชื่อมต่อคีย์แพดจริงเพื่ออัปเดตค่าแมโครบนอุปกรณ์ |
| **Export AHK** | ส่งออกแมโครเป็นสคริปต์ AutoHotkey v2 ซึ่งใช้ได้กับ Windows |

## 🛠️ วิธีแก้ไขปัญหาพื้นฐาน (Troubleshooting)
- **อุปกรณ์ไม่แสดง**: ตรวจสอบให้แน่ใจว่าใช้ Chrome/Edge เวอร์ชันล่าสุดและเปิดใช้งานการอนุญาต `WebHID` ใน Settings → Privacy → Site Settings → USB devices.  
- **ไฟล์ AHK ไม่ทำงาน**: ติดตั้ง AutoHotkey v2 จาก https://www.autohotkey.com/ แล้วคลิกสองครั้งที่ไฟล์ `macro.ahk`.  
- **ไม่มีเสียง**: ตรวจสอบว่าเสียงของระบบเปิดอยู่และไม่ได้บล็อกจากเบราว์เซอร์.  
- **ธีมไม่เปลี่ยน**: รีเฟรชหน้า (Ctrl+R) แล้วเลือกธีมอีกครั้ง.
